# Lightweight Personal Website Template
## Author: Kirsten Tempest

This repository contains a **lightweight, sustainability-first HTML template** for a personal or professional website.

It is designed to be:
- static (HTML + CSS only)
- easy to customise
- accessible by default
- low-carbon to host and run

The template is suitable for researchers, developers, designers, or anyone who wants a calm, minimal personal site without frameworks or build tools.

## How to use this template

1. **Copy or fork this repository** into your own GitHub or GitLab account.
2. Rename the repository to whatever you like.
3. Edit the HTML files:
   - You can customise the site by simply **finding and replacing text** in the HTML.
   - No prior HTML or CSS knowledge is required.
   - Replace names, paragraphs, links (e.g. LinkedIn, Google Scholar) with your own.
4. Optional: tweak colours or spacing in the single CSS file if you wish.
